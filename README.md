### 3. Simple compound intrest calculator
A simple Python program to calculate compound interest based on user input. This project is beginner-friendly and demonstrates the use of variables, input handling, and mathematical calculations in Python.
* Calculates final amount using compound interest formula
* Displays total compound interest earned
* Simple and easy-to-understand code
* No external libraries required

## 🧠 Formula Used

```
A = P(1 + r / (100 * n))^(n * t)
```

Where:

* **P** = Principal amount
* **r** = Annual interest rate (%)
* **t** = Time (in years)
* **n** = Number of times interest is compounded per year
* **A** = Final amount
*  [See Project](word_frequency_cal.py)
